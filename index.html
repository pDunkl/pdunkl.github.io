SDrawerPanel

DrawerPanel is a new slate widget, which is a swipeable bar (horizontal on bottom/top, vertical on left/right).
It can contain any widget to transition to any other widget.

In the video the small drawerpanel contains a listview, when swiping up it transitions to the full inventory.

InputDeviceProcessor

This class replaces the default InptuDeviceProcessor.
It detects, when a player (also local multiplayer) switches between different input methods.

KBM, Gamepad or Touch input, and then calls a delegate with that information.
Intended use is, when the game is played on mobile or PC or where the input methods aren't always the same.
Switching midgame from KBM to Gamepad etc. so the UI can change the layout accordingly to the new input method.

Complex unlit material to mimic shading using matcaps

Complex Material example, using matcaps as inside texture arrays, enabling different shading stylizes, while the material is just unlit. Intendet use was to make objects extremely customizeable with shading and colors for mobile. It uses Custom primitive data, so it wouldn't need dynamics material instances, and therefore same meshes can be batched for draw call reduction.
